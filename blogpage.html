Other Stuff

I especially want to bring your attention to this quote by Linus Torvalds:

Talk is cheap. Show me the code.
- Linus Torvalds

I like this quote a lot, because it reflects a major aspect of programming. Even if we think we know the perfect way to tackle a problem in programming, however perfect we may think it is, it is the code implementation that finally matters. We can never know how hard the code implementation is without trying to implement it. This is concisely said in his quote.
